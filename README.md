### 1. Interactive To-Do List (Level 2: Intermediate)

### 2. Full-Stack E-Commerce Website (Level 3: Advanced)
A robust, full-stack application simulating a real-world online shopping experience. This project demonstrates the integration of frontend, backend, and database.

- **Key Features:**
  - **User Authentication:** Secure Login and Signup functionality.
  - **Product Management:** Dynamic product gallery with filtering.
  - **Shopping Cart:** Add/remove items and calculate totals in real-time.
  - **Payment Integration:** Simulated secure checkout process.
- **Technologies Used (MERN Stack):** - **Frontend:** React.js
